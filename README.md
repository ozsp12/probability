# Probabilidade

Este repositório é dedicado a um curso de probabilidade completo, contendo os seguintes tópicos:
* Análise exploratória de dados, organização de dados, análise e teoria de erros, banco de dados.
* Axiomas de probabilidade, teoria de distribuições, probabilidade condicional, teorema de Bayes, análise frequentista.
* Métodos de estimação, inferência, mínimos quadrados, métodos de máxima verossimilhança, estimação por intervalos e testes de hipótese.
* Testes paramétricos e não-paramétricos.
* Modelos e distribuições, testes específicos, teorema do limite central e redução à média.
* Dados sintéticos e metodologia de sintetização de dados.
